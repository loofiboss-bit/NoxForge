# NoxForge KDE

NoxForge is an original dark, compact theme prototype for KDE Plasma 6. It uses
graphite surfaces, an electric-lime accent, restrained cyan and violet details,
and the angular Forge Notch signature.

The first milestone is intentionally narrow: a color scheme, a core Plasma
Style, an Aurorae decoration, 24 representative icons, one wallpaper, and safe
user-local tooling. Breeze remains the application style and supplies icon and
Plasma Style fallbacks where NoxForge intentionally has no asset.

## Status

The repository is being implemented through the `v0.1.0-design-prototype`
release gate. Graphical checks that require a live Plasma session are tracked as
pending until they are performed on Fedora KDE 44 Wayland at 100% and 140%
scaling.

See [the implementation plan](docs/IMPLEMENTATION_PLAN.md) for the locked scope
and [the compatibility record](docs/COMPATIBILITY.md) for verified Plasma 6
contracts.
